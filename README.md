# AI Chatbot API

AI-powered customer support chatbot built with Node.js, MongoDB, and OpenAI.

## Features

- JWT authentication
- Intent classification
- AI reply generation
- Conversation history
- Escalation rules
- MongoDB storage

## Tech

Node.js
Express
MongoDB
Mongoose
OpenAI API
JWT

## Endpoints

POST /api/auth/register  
POST /api/auth/login  
POST /api/chat  
GET /api/chat/:id
